# TrAitement_DoQment
TrAitement-DoQment turns scanned PDFs and images into a fully local, searchable knowledge base. Users can ask natural-language questions across all documents and get answers traceable to the exact source and location, without any data leaving their machine.

Téléchargez la base de donnés dans: https://drive.google.com/drive/folders/1ShItNWXyiY1tFDM5W02bceHuJjyeeJl2 puis la dezip dans data/

run ingestion.py
